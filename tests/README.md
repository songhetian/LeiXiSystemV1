# 测试文件夹

这个文件夹包含所有的测试脚本和测试文件。

## 测试脚本

### test-db.js - 数据库连接测试

测试数据库连接是否正常。

**运行方式：**
```bash
node tests/test-db.js
```

**功能：**
- 测试数据库连接
- 验证配置是否正确
- 检查数据库是否存在
- 提供错误诊断信息

**输出示例：**
```
测试数据库连接...
配置: { host: 'localhost', user: 'root', ... }
✅ 数据库连接成功！
✅ 查询测试成功
✅ 数据库存在: leixin_customer_service
```

## 添加新测试

在这个文件夹中创建新的测试文件，命名规范：
- `test-*.js` - 单元测试
- `integration-*.js` - 集成测试
- `e2e-*.js` - 端到端测试

## 运行所有测试

```bash
# 运行数据库测试
node tests/test-db.js

# 未来可以添加更多测试
# npm test
```

---

**注意：** 测试文件不会被打包到生产环境中。
